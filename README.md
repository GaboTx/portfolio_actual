# Gabriel Sareñana — Portfolio

Personal portfolio site. Built with vanilla HTML, CSS and JavaScript — no frameworks, no build step.

**Live:** [gabotx-portfolio.netlify.app](https://gabotx-portfolio.netlify.app)

## Structure

```
portfolio/
├── index.html        # Main page
├── css/
│   └── style.css     # All styles
├── js/
│   └── main.js       # Interactions & animations
└── netlify.toml      # Deployment config
```

## Deploy

Push to GitHub. Connect the repo in [Netlify](https://app.netlify.com) and it auto-deploys on every push to `main`.

No build command needed — publish directory is `.` (root).
